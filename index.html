Advisor Bot

Advisor bot is a command-line run program written in C++ that provides users with data pertaining to the trade of currencies. Upon the initialization of the program, a CSV file is parsed by the advisor bot, extracting values for timestamps, products, order types, prices, and amounts. Using this data, the program is able to run various functions such as calculating statistical values that include minimum, maximum, and average values as well as predicting values for future timestamps.
